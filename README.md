# Spendit

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 15.1.4.

## Project's URL

It's [https://app.romanenko.co](https://app.romanenko.co/#)

## Package manager

Used `npm`

## Versions

My setup is:

- `npm -v` `9.4.1`

- `node -v` `v18.12.1`

It should work at every node/npm version that supports Angular 15.x

## Development server

Run `npm run start` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.

## Build

Run `npm run build` to build the project. The build artifacts will be stored in the `dist/spendit` directory.
